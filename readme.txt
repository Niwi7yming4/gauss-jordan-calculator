# 🧮 高斯-約旦消去法計算器

一個響應式的網頁應用程式，用於計算線性方程組的高斯-約旦消去法，支持分數計算和詳細的步驟顯示。

---

## ✨ 功能特色

- 🎯 **精確計算**: 支持分數運算，避免浮點數誤差
- 📱 **響應式設計**: 完美適配手機、平板和電腦
- 🎨 **數學風格界面**: 優雅的數學符號和視覺設計
- 📊 **逐步顯示**: 詳細展示計算過程的每一步
- 🔢 **範例矩陣**: 一鍵填入測試範例
- 💾 **靜態部署**: 無需後端，純前端實現

## 📖 使用說明

### 基本操作
1. **設置矩陣大小**: 選擇方程組的變數數量 (2-6)  
2. **輸入矩陣元素**: 在表格中輸入係數矩陣和常數項  
3. **選擇計算模式**: 勾選「使用分數計算」獲得精確結果  
4. **計算結果**: 點擊 "🧮 計算" 按鈕執行運算  
5. **查看過程**: 在結果區域查看詳細的計算步驟  

### 輸入格式
- 整數: `4`, `-2`, `11`  
- 分數: `1/2`, `-3/4`, `5/7`  
- 小數: `0.5`, `-0.75` (不推薦，建議使用分數)

📝 數學原理
高斯-約旦消去法

高斯-約旦消去法是一種求解線性方程組的算法，通過以下步驟將增廣矩陣化簡為行最簡形式：

1. **前向消去**: 將矩陣化為上三角形式 
2. **後向消去**: 將上三角矩陣化為對角形式  
3. **正規化**: 使對角線元素為 1
